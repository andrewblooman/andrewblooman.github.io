Personal site for [Andrew Blooman](https://andrewblooman.github.io) — Cloud Security Engineer based in London.

## Stack

Plain HTML, CSS, and vanilla JavaScript. No build step, no framework, no dependencies.

## Local preview

```bash
python3 -m http.server 8080
```

Then open [http://localhost:8080](http://localhost:8080).

## Deploy

Push to `main` — GitHub Pages publishes automatically.
